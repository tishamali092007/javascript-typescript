# 🌤️ WeatherX - Weather Forecast Application

A modern and responsive **Weather Forecast Web Application** built using **HTML, CSS, and JavaScript**. The application provides real-time weather information, current location weather, and a 5-day weather forecast using the OpenWeatherMap API.

---

# 📌 Project Objective

The objective of this project is to develop a Weather Forecast Application that allows users to search weather information for any city around the world. The application displays current weather details, a 5-day forecast, and supports fetching weather based on the user's current location.

---

# 🚀 Features

- 🌍 Search Weather by City Name
- 📍 Current Location Weather
- 🌡️ Real-Time Temperature
- ☁️ Weather Condition
- 💧 Humidity
- 💨 Wind Speed
- 👁️ Visibility
- 🌅 Sunrise Time
- 📅 Current Date
- 📆 5-Day Weather Forecast
- 🎨 Dynamic Background Based on Weather
- ⏳ Loading Animation
- 📱 Fully Responsive Design

# ⚙️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Open **index.html** in your browser.
4. Enter any city name or use the location button.
5. View the current weather and 5-day forecast.

---

# 🌐 API Used

**OpenWeatherMap API**

Features used:

- Current Weather API
- 5-Day Forecast API
- Weather Icons API

# 💡 Functionalities

### Search Weather

- Search weather by city name.
- Press **Enter** or click the Search button.

---

### Current Location

- Click the **Location** button.
- Allow browser location permission.
- Weather details will be fetched automatically.

---

### Weather Information

The application displays:

- City Name
- Current Date
- Temperature
- Weather Condition
- Humidity
- Wind Speed
- Visibility
- Sunrise Time

---

### 5-Day Forecast

Displays:

- Day Name
- Weather Icon
- Temperature

---

### Dynamic Background

The application automatically changes its background according to weather conditions:

- ☀️ Clear
- ☁️ Clouds
- 🌧️ Rain
- ❄️ Snow
- ⛈️ Thunderstorm
