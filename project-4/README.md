# 🧭 Adventure Quiz Application

## 🔹 Project Description

Adventure Quiz Application is a modern and interactive quiz application built using **HTML**, **CSS**, and **JavaScript**.

The application presents users with adventure and geography-related multiple-choice questions in a clean and attractive interface. Each question includes a countdown timer, progress indicator, and interactive answer selection to create an engaging quiz experience.

At the end of the quiz, users receive their final score along with the percentage of correct answers and can restart the quiz anytime.

---

## 🔹 What is Included in This Project

* 10 Adventure & Geography Quiz Questions
* Multiple Choice Answer Cards
* 10-Second Countdown Timer
* Progress Dots Indicator
* Question Counter
* Interactive Answer Selection
* Automatic Question Change on Timer End
* Final Score & Percentage Display
* Restart / Play Again Functionality
* Modern Glassmorphism UI
* Fully Responsive Design

## 🔹 Project Features

### ❓ Interactive Quiz

Users answer multiple-choice questions through an attractive card-based interface.

### ⏱️ Countdown Timer

Each question includes a **10-second timer**.

If the timer reaches zero, the quiz automatically moves to the next question.

### 📊 Progress Indicator

Displays:

* Current Question Number
* Progress Dots
* Completed Questions

### ✅ Answer Selection

Users can select only one answer for each question before moving to the next question.

### 🏆 Final Result

After completing all questions, users can view:

* Total Score
* Percentage
* Quiz Completion Message

### 🔄 Restart Quiz

Users can restart the quiz anytime using the **Restart** or **Play Again** button.

### 📱 Responsive Design

The application is optimized for:

* Desktop
* Tablet
* Mobile Devices
