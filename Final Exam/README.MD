# 📝 Tasky – Task Manager

A modern and responsive **Task Management System** built using **HTML, CSS, and JavaScript**. This project allows users to manage their daily tasks with full CRUD functionality, priority filtering, search, and Local Storage support.

---

## 📌 Project Objective

The objective of this project is to develop a Task Management System where users can efficiently organize their tasks by performing CRUD operations (Create, Read, Update, Delete). The application also includes priority-based filtering, task searching, form validation, and Local Storage to save data permanently in the browser.

---

# 🚀 Features

- ✅ Create New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- 🔍 Search Tasks by Title or Description
- 🎯 Filter Tasks by Priority
  - All
  - Low
  - Medium
  - High
- 📅 Due Date Management
- ⚠️ Overdue Task Detection
- 💾 Local Storage Support
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism UI
- 🔔 Toast Notifications
- ✔️ Form Validation

# ⚙️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Open **index.html** in your browser.
4. Start creating and managing your tasks.

No installation is required.

---

# 💡 Functionalities

### Create Task
- Enter Task Title
- Enter Description
- Select Priority
- Select Due Date
- Click **Create Task**

---

### Update Task

- Click the Edit icon.
- Modify task details.
- Click **Update Task**.

---

### Delete Task

- Click the Delete icon.
- Confirm deletion.
- Task is permanently removed.

---

### Search

Use the search box to find tasks by:

- Task Title
- Task Description

---

### Priority Filter

Filter tasks using:

- All
- Low
- Medium
- High

---

### Local Storage

All tasks are automatically stored in the browser using **Local Storage**, so the data remains available even after refreshing or reopening the browser.
