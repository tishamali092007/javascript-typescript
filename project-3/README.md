 ⏳ Countdown & Quotes Application

 🔹 Project Description

Countdown & Quotes Application is a simple and interactive web application built using HTML, CSS, and JavaScript.

This project combines a live countdown timer with motivational quote slides. Users can track the remaining time until a specific event while also viewing inspiring quotes through a smooth slider interface.

The application includes both automatic and manual slide navigation, making the experience engaging and user-friendly.

---
 🔹 What is Included in This Project

* Live Countdown Timer
* Days, Hours, Minutes & Seconds Display
* Pause & Resume Countdown Functionality
* Motivational Quote Slider
* Previous & Next Navigation Buttons
* Auto Slide Functionality
* Welcome Modal Popup
* Smooth Transition Effects
* Responsive Design for Mobile & Tablet Devices

🔹 How This Project is Made

### HTML

HTML is used to create the basic structure of the application, including:

* Countdown Section
* Quote Slides
* Navigation Controls
* Welcome Modal

### CSS

CSS is used to design and style the application.

Features include:

* Flexbox Layout
* CSS Grid Layout
* Responsive Design
* Gradient Backgrounds
* Smooth Transitions
* Modern User Interface

### JavaScript

JavaScript is used to add interactivity and dynamic functionality.

It handles:

* Countdown Calculations
* Timer Updates Every Second
* Pause & Resume Logic
* Slide Navigation
* Automatic Slide Changes
* Welcome Modal Display
